# Faker.lbaction

1. Install `faker-cli` by `composer global require bit3/faker-cli`
2. Clone the repo to `$HOME/Library/Application Support/LaunchBar/Actions`
3. Type <kbd>Faker</kbd><kbd>Tab</kbd> and chose one of the suggestions
